adversary simulation tool

An adversary simulation tool written in C# .NET compiled with .NET 3.5 for windows 7 and .NET 4.5 for windows 10, consists of a dropper , a watchdog , and the final payload.
Some of it's features: AV evasion persistance, chrome decryptor , keyloagger, command line and powershell shell session, in memory execution of payloads and c# assemblies,sneaky files and data exfiltration.
This is a (very) minimalistic version of the dropper and payload, mainly aimed at windows 7 and .NET framework 3.5 :
Project link : github.com/khraoverflow/iruwdDead/
The dropper: github.com/khraoverflow/iruwdDead/iruwd7
The payload: github.com/khraoverflow/iruwdDead/explorer
